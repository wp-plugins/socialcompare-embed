=== SocialCompare embed ===Contributors: socialcompareTags: socialcompare, embed, comparison, table, matrix, compare, gridRequires at least: 2.9Tested up to: 3.1.1Stable tag: trunkAllows to easily embed a comparison table from SocialCompare within a post or page.== Description ==Allows to easily embed a comparison table from SocialCompare, within a post using shortcode, or oembed.You can use directly the comparison url or the embed url.Syntax:* `[socialcompare]http://socialcompare.com/en/comparison/apples-and-oranges[/socialcompare]`* `[sc height="960"]http://socialcompare.com/en/w/apples-and-oranges[/sc]`* `http://socialcompare.com/en/comparison/apples-and-oranges (URL only on a single line)`== Installation ==1. Upload SocialCompare plugin to the `/wp-content/plugins/` directory1. Activate the plugin through the 'Plugins' menu in WordPress1. Use the shortcode within your posts== Frequently Asked Questions === Can I create a comparison table directly from my blog? =No you can only use comparison tables available on [SocialCompare](http://socialcompare.com/ "User-generated comparisons tables") website.You can easily use existing comparison matrix available at SocialCompare.If you'd like to create your own table you can create a free account on SocialCompare.= Can I customize the design of the table? =Yes embedded tables are easily customizable, even if you are not the creator of the table.You can customize font-size, colors, available actions (sort/hide...)...== Screenshots ==1.
